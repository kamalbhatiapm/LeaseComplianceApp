🔴 Critical gaps to fix before Beta

1.Binary mismatch: Frontend sends base64 JSON — n8n expects binary file attachment. Fix: send file as multipart/form-data OR extract text client-side and send as plain text.
2.Missing intent field: n8n Orchestrator reads body.intent which is never set by the frontend. Fix: add intent: "key_term_extraction" to the POST payload.
3.Response schema mismatch: n8n returns {contract_type, key_terms[]} but frontend expects {risk_score, flags[], terms_found[]}. Fix: align schemas per BACKEND-PLAN.md Phase 1.
4.PDF clause viewer (BUG-006): Clause citations render but open nothing. Hard auditor acceptance blocker.
5.No persistence (BUG-009): Page refresh destroys all results. Supabase client never instantiated in browser despite credentials being injected.
6.PDF export not wired: Report gate works correctly but the actual PDF is never generated.
